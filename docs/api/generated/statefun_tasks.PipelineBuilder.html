inline(is_inline=True)¶

Marks the pipeline as being inline (or not). By default pipelines are not inline. Inline pipelines accept inputs from and share state with their parent task. :rtype: PipelineBuilder :return: the builder

  • events (option) – event handler instance if this pipeline should fire events
